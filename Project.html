The project "Snake Game with Deep Q-Learning" implements the classic Snake game using the Deep Q-Learning (DQN) algorithm. It includes components for training the agent, evaluating its performance, and running the game. Key features include:
 - Training: Train the agent using the DQN algorithm.
 - Game Play: Play the game manually using the      keyboard.
 - Evaluation: Evaluate the trained agent's                performance after training.

The Student Management System Project is built using C++ and allows users to manage student data. Key features include:
 - Add, Edit, and Delete Student Information: Users can input new student details, update existing records, or remove students from the system.
 - View Student Grades: The system allows for easy viewing of student grades and performance.
 - Data Management: A simple and efficient interface for managing large datasets related to student records.
This project helps demonstrate the usage of basic file handling, class structures, and user interaction in C++.
